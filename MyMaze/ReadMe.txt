Приветствую
Я написал небольшой код рисующий лабиринты
мною реализовано 2 метода построения
DFS и случайный алгоритм Прима (по сути метод поиска минимального остовного древа)
программа использует модуль pygame для визуализации (py -m pip install -U pygame --user - команда для установки)
при запуке да экране появляется окно для отрисовки лабиринта а в консоли небольшая инструкция
Hello!
Press Q or E to switch mod
Press W to read last Maze
Press SPACE to start drawing
Press F to draw a right way
Enjoy!
После этого по нажатию пробела лабиринт начнет отрисовыватся (если была нажата q то методом Прима если e то DFS)
F покажет правильный путь (по умолчанию от правого нижнего угла до левого верхнего)
функция поиска пути может соединить любые две точки в лабиринте но как аргументы для модификации я их не вынес
Также код умеет записывать и читать лабиринты из файлов
в папку с игрой он записывает последний нарисованый лабиринт в файл "LastMaze"
по нажатию w он его считает

пример работы программы:
1)запустить из терминала командой py MazeGenerator.py
2)нажать пробел
3)подождать отрисовки и нажать F
4)при желании нажать пробел еще раз
в папке есть примеры работы кода
P.s. Кнопки иногда чуть залипают и может понадобится повторное нажатие
